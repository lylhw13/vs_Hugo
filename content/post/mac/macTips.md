# shortcuts
- `control + a` to go beginning of a line
- `control + e` to go end of a line

# terminal
```sh
# calculate md5 checksum
md5 /tmp/filename
# calculate SHA-1 checksum
shasum -a 1 filename
# calculate SHA-256 checksum
shasum -a 256 filename
```
