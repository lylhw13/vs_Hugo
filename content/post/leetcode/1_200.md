## 191. Number of 1 Bits
```c++
class Solution {
public:
    int hammingWeight(uint32_t n) {
        int count = 0;
        while (n > 0) {
            n = n & (n - 1); //Remove the lowest order 1 bit
            count++;
        }
        return count;
    }
};
```
